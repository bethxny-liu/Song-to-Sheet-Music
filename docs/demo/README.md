# Demo assets

Example conversion for the README: **Mary Had a Little Lamb** (single staff, 90 BPM).

| File | Description |
|------|-------------|
| `mary-had-a-little-lamb.mp3` | Input audio |
| `pitch-chart.png` | Detected pitch over time |
| `sheet-music.png` | Sheet music preview |
| `mary-had-a-little-lamb.pdf` | Downloadable PDF export |
